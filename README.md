# Sistema de Control de Acceso - Colegio Las Mercedes

Este proyecto es una aplicación móvil desarrollada en **Android Studio** con **Kotlin** para gestionar el ingreso de personal en una institución educativa.

### 🚀 Características
* Registro de ingreso de docentes y personal administrativo.
* Escaneo de DNI para un control rápido y seguro.
* Base de datos local para el historial de accesos.

### 🛠️ Tecnologías
* Kotlin / Java
* SQLite / Room
* Android Jetmark (Fragments, ViewBinding)
